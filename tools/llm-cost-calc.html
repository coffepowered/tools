ℹ️ How Caching is Calculated

Cache Mechanism: Prompt caching stores frequently reused input tokens (like system prompts, instructions, or common context) to reduce costs and latency on subsequent requests.

  • OpenAI: Automatic caching. Cache reads receive a 90% discount (10% of input price). No cache write surcharges.
  • Anthropic: Manual cache control with 90% discount on cache reads (10% of input price). Cache writes cost 25% more than input.
  • Google Gemini: Context caching with 90% discount on cache reads (10% of input price) plus a small storage fee for active caches.
  • Z.ai / Zhipu AI (GLM): Automatic prompt caching with up to ~80% discount on cache reads. No cache write surcharges.
  • Mistral: No prompt caching currently available.

In this calculator: When you set cache usage to 50%, it means 50% of your input tokens are read from cache (at discounted rate), and 50% are processed fresh. For Anthropic models, cache writes incur a one-time surcharge of 25% on the cached inputs.
